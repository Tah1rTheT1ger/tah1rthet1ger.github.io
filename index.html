Projects

Projects

Here are some of the exciting projects I've built using C, C++, Java, Python, and a variety of other tools and frameworks during my awesome journey at IIIT-B!

Flexcellent - Yoga Pose Correcting Application

A Yoga Pose Corrector app that uses computer vision to detect poses in real-time, compares them to ideal references, and provides instant feedback to improve their form. It uses OpenCV and mediapipe to help users correct postures efficiently, with audio feedback for hands-free use. Used websockets for server-client interaction.

Tools Used: Flutter, Dart, Python, OpenCV

Fruit Detection System for the Visually Impaired

A Fruit Detection System designed for the visually impaired, using deep learning and computer vision to identify twelve fruits. It incorporates hand gesture tracking and voice interaction for a completely touch-free experience, with ReactJS and Streamlit front-ends providing accessible interfaces with text-to-speech functionality.

Tools Used: Python, OpenCV, Tensorflow, Keras, ReactJS

Online Library Management System - OLMS

Developed a multithreaded OLMS. The server has features like file locking to prevent race conditions, concurrency handling for multiple clients, and password authentication. It also has library functions for adding, searching, updating, and deleting books. The client-side connects to the server and browse, borrow available books.

Tools Used: Linux, C

Image Effects Web Application

Developed a Web Application that applies various image effects to JPEG or PNG images, with a logging system. Users can choose effects such as Brightness, Contrast, Blur, Greyscale, Sepia, and more, with the option to download the edited image. The effects are implemented in C++ and accessed via a Java wrapper using JNI.

Tools Used: Java, C++, JNI

MIPS Processor Simulator

Developed a MIPS assembler and processor simulation in Python. The assembler converts MIPS assembly language to machine code, and the processor simulates execution. It includes instruction fetching, decoding, execution, memory access, and write-back stages, with non-pipelined and pipelined versions implemented separately.

Tools Used: Python

MasterMind using Digital Logic

Developed a working version of the board game "MasterMind" a.k.a "Cows and Bulls" or "Code Breaker." We implemented it on CircuitVerse using combinatorial circuits with buffers, counters, flip-flops, and logic gates. The game's logic was built from scratch, including random code generation, guess checking, and feedback generation, with full functionality.

Tools Used: CircuitVerse
